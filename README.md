Выпускная квалификационная работа на тему "Разработка интеллектуальной системы автоматической генерации экзаменационных билетов с использованием LaTeX"

Файлы в корне проекта ticket_generator.py и requirements.txt относятся к первой версии проекта, а директория examticketgeneratorweb-main содержит новую версию проекта,
скопированную из моего другого репозитория: https://github.com/ARMIZKA/examticketgeneratorweb
